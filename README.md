# EnTranslate的由来

## 简洁
EnTranslate是一款纯粹的滑词翻译软件
1. 支持滑词翻译
2. 支持调用百度翻译接口翻译注释文档
3. 强大的单词拆分能力: 支持驼峰, 下划线形式等各种单词拆分
4. 丰富的本地词库: 包含 340 万+离线单词, 支持各种生僻单词

EnTranslate内部实现方法借鉴于vscode插件[Code Translate](https://github.com/w88975/code-translate-vscode) ，感谢w88975作者将其开源。
